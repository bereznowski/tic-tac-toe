# Tic-Tac-Toe

This is a simple terminal-based game called [Tic-Tac-Toe](https://en.wikipedia.org/wiki/Tic-tac-toe).

To compile it on GNU/Linux use:

```
g++ -o main main.cpp draw_grid.cpp choose_square.cpp check_win.cpp -lcurses
```
